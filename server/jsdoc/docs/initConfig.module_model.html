Type Definitions

InitConfigSchema

Esquema para gestionar la configuración de inicialización de la base de datos. Este esquema permite registrar diferentes estados de inicialización.
Type:
  • Object
Properties:
Name Type Description
key string Clave única que indica el estado de inicialización. Puede tomar valores como 'routes_initialized' o 'address_types_initialized'.
initialized boolean Indica si la configuración ha sido inicializada.
date Date Fecha de creación del registro.
