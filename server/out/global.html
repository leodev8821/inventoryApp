(async) getTownsInfo(link) → {Promise.<Array.<{municipio: string, provincia: string, cod_INE: string}>>}

Extrae información detallada de los municipios desde una tabla específica en https://www.ign.es/web/ane-datos-geograficos/-/datos-geograficos/datosPoblacion?tipoBusqueda=municipios.
Parameters:
Name Type Description
link string URL de la página que contiene la tabla con la información de los municipios.
Source:
  • scraping.util.js, line 3
Throws:
- Lanza un error si ocurre un problema durante el proceso de scraping o si la estructura de la página no coincide con lo esperado.
Type
Error
Returns:
- Una promesa que resuelve en un array de objetos, donde cada objeto contiene el nombre del municipio, la provincia y el código INE.
Type
Promise.<Array.<{municipio: string, provincia: string, cod_INE: string}>>
Example
const townsInfo = await getTownsInfo('https://www.ign.es/web/ane-datos-geograficos/-/datos-geograficos/datosPoblacion?tipoBusqueda=municipios');
console.log(townsInfo);

(async) scrapeSpainsTowns() → {Promise.<Array.<{municipio: string, provincia: string, cod_INE: string}>>}

Realiza scraping de la página del IGN para obtener información detallada sobre los municipios de España.
Source:
  • scraping.util.js, line 61
Throws:
- Lanza un error si ocurre un problema durante el proceso de scraping, incluyendo errores al obtener la información de la página del IGN o si la estructura de la página no coincide con lo esperado.
Type

Returns:
- Una promesa que resuelve en un array de objetos, donde cada objeto contiene el nombre del municipio, la provincia y el código INE.
Type
Promise.<Array.<{municipio: string, provincia: string, cod_INE: string}>>
